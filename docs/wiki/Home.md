# Universal Downloader Wiki

Universal Downloader is a native Android and iOS app for accepting shared media links and sending them through a downloader pipeline.

## Pages

- [Release Process](Release-Process.md)
- [Testing and Coverage](Testing-and-Coverage.md)
- [Product Roadmap](Product-Roadmap.md)
- [Troubleshooting](Troubleshooting.md)

## Current Priorities

- Keep every PR protected by release-variant tests and e2e tests.
- Publish signed Android APKs only through the manual Release workflow.
- Keep coverage visible through GitHub Actions artifacts.
- Decide the production downloader runtime for iOS.
